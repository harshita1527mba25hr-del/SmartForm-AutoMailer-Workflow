SmartForm-AutoMailer-Workflow

## Project Overview
SmartForm-AutoMailer-Workflow is an automated system that collects responses from Google Forms, stores them in Google Sheets, and sends personalized email notifications using Gmail.

It eliminates manual work, ensures real-time communication, and improves efficiency.

---

##  How It Works

1. User fills out the Google Form  
2. Responses are stored in Google Sheets  
3. n8n detects new row entry (Trigger)  
4. Gmail sends an automatic email  

---

##  Tech Stack

- Google Forms  
- Google Sheets  
- Gmail  
- n8n  

---

##  Workflow Architecture

```
Google Form → Google Sheets → n8n Trigger → Gmail → User
```

## Setup Instructions

1. Create a Google Form  
2. Link it to Google Sheets  
3. Open n8n  
4. Add Google Sheets Trigger (Row Added)  
5. Add Gmail Node (Send Email)  
6. Connect your Google account  
7. Customize the email message  
8. Activate the workflow
